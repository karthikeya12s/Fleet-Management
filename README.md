# Fleet Management — Product Management Learning Kit

A comprehensive, hands-on learning kit for mastering product management through the lens of fleet management SaaS. Includes business model analysis, PRDs grounded in real-world problems, structured exercises mapped to a 6-phase learning roadmap, and 7 interactive browser-based tools.

## Repository Structure

```
├── docs/            # Learning overview and roadmap
├── prds/            # Product Requirements Documents (5 business models)
├── exercises/       # PM exercises aligned to the learning roadmap
└── tools/           # Interactive HTML tools (open in browser)
```

## Documents

### Learning Foundation (`docs/`)
| File | Description |
|------|-------------|
| `fleet-management-learning-overview.md` | Industry landscape, technology stack, business models, and tooling ecosystem |
| `fleet-management-learning-roadmap.md` | 6-phase structured roadmap: Foundation → Technical Stack → Domain Expertise → Advanced Topics → Product Strategy → Hands-On Application |

### Product Requirements Documents (`prds/`)
| PRD | Product | Business Model | Target Problem |
|-----|---------|---------------|----------------|
| PRD-01 | **FleetPulse** | SaaS Platform | SMB fleets managing operations with spreadsheets |
| PRD-02 | **FleetNode** | Hardware + Software Bundle | $74B annual truck downtime from fragmented telematics |
| PRD-03 | **FleetBridge** | Managed Services | 20-30% TCO overspend from in-house fleet management |
| PRD-04 | **FleetSignal** | Data Monetization | Untapped fleet telemetry value for insurers, cities, OEMs |
| PRD-05 | **FleetBay** | Marketplace | 12-18 hrs/week wasted on vendor sourcing |

### Exercises (`exercises/`)
Each file contains 16-20 exercises mapped to the 6-phase learning roadmap, covering market sizing, competitive analysis, technical architecture, pricing strategy, GTM planning, and more.

## Interactive Tools (`tools/`)

Open `tools/index.html` in a browser for the full hub, or launch individually:

| Tool | Description | Mapped Exercise |
|------|-------------|-----------------|
| **Fleet TCO Calculator** | Cost-of-ownership modeling with SaaS ROI analysis | Ex 1.4, 1.3 |
| **Unit Economics Simulator** | LTV, CAC, margins across 5 business models | Phase 1 (all models) |
| **RICE Prioritization Board** | Score, rank, and export feature backlogs | Ex 3.3 (SaaS) |
| **Marketplace Simulator** | GMV, take-rate, liquidity, metro expansion | Ex 1.1 (Marketplace) |
| **Vehicle Replacement Analyzer** | Optimal replacement point via TCO/mile curves | Ex 3.3 (Managed Services) |
| **Competitive War Room** | Feature matrix, radar, SWOT, positioning map | Ex 1.2, 5.3 |
| **Sprint Planning Simulator** | Drag-and-drop sprint board with velocity tracking | Ex 6.1 (SaaS) |

All tools are self-contained HTML files — no build tools, no dependencies. Just open in a browser.
